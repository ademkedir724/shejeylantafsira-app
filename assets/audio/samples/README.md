# Sample Audio Files

This directory contains the 3 bundled LQ audio files for sample pages (1, 2, and 604).

These files are shipped inside the app binary (~4MB total) so users can hear Sheikh Jeylan's tafsir immediately after install without downloading anything.

## Files

- `1.mp3` — Page 1 (Al-Fatiha)
- `2.mp3` — Page 2 (Al-Baqarah, beginning)
- `604.mp3` — Page 604 (An-Nas)

## Source

All audio files are recordings of Sheikh Jeylan's Afaan Oromo tafsir.
Full LQ and HQ audio for all 604 pages is hosted on Cloudflare R2 remote storage.
